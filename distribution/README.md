# YN WPS 中印双语翻译插件

版本：1.0.0

## 适用环境

- 仅支持 Windows 版 WPS 表格。
- 建议 WPS Office 12.1.0.16910 或更高版本。
- 默认使用智谱开放平台 GLM-4-Flash 大模型（免费）翻译，兼容 OpenAI 接口协议。
- 每位员工使用自己的智谱 API Key；如需可在设置中改用其他兼容 OpenAI 协议的服务（Gemini、硅基流动等）。

## 安装

1. 关闭所有 WPS Office 窗口。
2. 双击 `install.cmd`。
3. 安装成功后重新打开 WPS 表格。
4. 功能区会出现“中印翻译”选项卡。

安装仅作用于当前 Windows 用户，不需要管理员权限。

## 首次配置

1. 打开 WPS 表格，进入“中印翻译”选项卡。
2. 点击“API 设置”。
3. 填写智谱 API Key（智谱开放平台 → 个人中心 获取）。
4. 点击“保存并测试”。
5. 按需要调整请求间隔、已有双语内容替换和自动备份选项。

密钥由 Windows DPAPI 按当前用户加密保存，不会写入工作簿、插件脚本或日志。

## 翻译逻辑

- 输出格式为：第一部分保留原文，换行后写入目标语种译文。
- 支持翻译选区、当前工作表和整个工作簿。
- 支持中文、英文、印尼语互译，源语种自动识别，目标语种在“API 设置”中选择。
- 自动跳过公式、数值、日期和空白单元格。
- 合并单元格只处理左上角单元格，不拆分合并区域。
- 保留原字体、边框、颜色、对齐、合并和列宽。
- 自动启用换行；普通单元格自动适应行高，合并单元格保守增高。
- 默认允许重新翻译并替换已有双语内容。
- 默认在原文件旁创建“翻译前备份”副本。
- 相同文本使用本机缓存，减少 API 调用。

## 专业术语

点击“术语表”，每行输入一个固定译法：

```text
上岗培训=Pelatihan sebelum bertugas
一线岗位=Posisi lini depan
```

术语表按目标语种分开保存；翻译时优先使用当前目标语种术语表中的译法。

## 卸载

1. 关闭所有 WPS Office 窗口。
2. 双击 `uninstall.cmd`。

默认保留加密 API 设置、术语表和缓存。需要彻底清理时，可执行：

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\uninstall.ps1 -RemoveUserData
```

## 故障排查

- 看不到“中印翻译”：确认安装时已关闭 WPS，然后重新启动 WPS。
- 提示助手未运行：双击 `start-helper.cmd`，或从开始菜单启动翻译助手。
- 提示鉴权失败：检查 API Key 是否正确。
- 提示请求过于频繁：在“API 设置”中增大请求间隔。
- 提示额度/余额不足：检查智谱账户的免费额度或余额。
- 日志位置：`%LOCALAPPDATA%\YNWpsTranslator\helper.log`。
